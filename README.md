  1.2、spring-boot-starter-aop
2、mybatis （数据库持久化框架）  
3、mybatis-plus （mybatis 自动 crud 功能）
4、alibaba druid （数据库连接池）
5、springfox-swagger （ Automated JSON API documentation for API's built with Spring）
6、lombok （简化javabean代码 工具）
7、fastjson （json 处理）
